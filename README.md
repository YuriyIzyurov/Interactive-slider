# React Interactive History dates slider
## Deployed app
https://interactive-sliderrrrr.vercel.app/
## Installation
1. Clone/download repo
2. `yarn install` (or `npm install` for npm)

## Usage
**Development**

`yarn run start-dev`

* App served @ `http://localhost:8080`

**Production**

`yarn run start-prod`

* App served @ `http://localhost:3000`
